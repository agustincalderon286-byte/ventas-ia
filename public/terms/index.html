Terminos

Reglas claras para usar Agustin 2.0.

Estos terminos aplican al uso de Agustin 2.0 Chef y al programa de mensajes de seguimiento de Agustin 2.0 para personas que pidieron informacion, recetas, ayuda o contacto.

Vigente desde el 25 de marzo de 2026 Programa: Agustin 2.0 Messages

Descripcion del programa

Agustin 2.0 ofrece un asistente digital para recetas, cocina mexicana saludable, agua, productos del hogar y seguimiento a solicitudes de informacion. El programa de mensajes se usa para compartir acceso, dar seguimiento, responder dudas y ayudar con citas o soporte cuando la persona lo solicito.

Frecuencia de mensajes y tarifas

  • La frecuencia de mensajes varia segun la interaccion, solicitud o seguimiento.
  • Pueden aplicar tarifas de mensajes y datos segun el plan movil del usuario.
  • El consentimiento para recibir mensajes no es condicion para realizar una compra.

Como salir o pedir ayuda

  • Para dejar de recibir mensajes, responde STOP.
  • Para recibir ayuda, responde HELP.

Uso permitido

El usuario acepta usar Agustin 2.0 de manera licita y respetuosa.

  • No usar el servicio para actividades ilegales, fraudulentas o abusivas.
  • No intentar acceder sin autorizacion a sistemas, cuentas o datos.
  • No enviar informacion falsa o solicitudes engañosas a traves del servicio.

Disponibilidad y cambios

Agustin 2.0 puede actualizar sus funciones, contenido o procesos con el tiempo. El servicio puede cambiar, pausarse o mejorarse sin previo aviso para mantener seguridad, calidad y operacion.
